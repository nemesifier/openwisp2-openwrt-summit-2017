That day, I finally understood what, in my humble opinion, is the most important reason for which Open Source software is so convenient for companies.

It's because it allows companies to share resources with one another to fix the low level common problems, those problems that the general public usually considers boring while geeks love to work on.

By adopting open source solutions, companies can deliver value without compromising their core business, thus avoid the hard work of developing their own solution from scratch.

The resources saved in reinventing the wheel can be used to improve the adopted solution, creating a virtuous cycle.

Enter OpenWISP 2

the modular configuration manager for OpenWRT / LEDE

Which you can reuse instead of building
your own from scratch.
